DownloadCallbackManager

The DownloadCallbackManager class is responsible for managing the download completion and error handling callbacks. It stores the callbacks and provides methods for adding and removing callbacks based on a unique identifier.

Properties

private var downloadCompletionCallbacks: [String: DownloadFinishedCallback]

A dictionary to store download completion callbacks with unique identifiers as keys.

Methods

func addDownloadFinishedCallback(_ callback: @escaping DownloadFinishedCallback, forIdentifier identifier: String)

Adds a download completion callback to the downloadCompletionCallbacks dictionary with the specified identifier.

func removeDownloadFinishedCallback(forIdentifier identifier: String)

Removes the download completion callback associated with the specified identifier from the downloadCompletionCallbacks dictionary.

func callDownloadFinishedCallback(forIdentifier identifier: String, withLocation location: URL?)

Calls the download completion callback associated with the specified identifier and passes the downloaded file location as a parameter. The callback is removed from the downloadCompletionCallbacks dictionary after it is called.
