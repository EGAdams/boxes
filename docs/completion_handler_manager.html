CompletionHandlerManager

The CompletionHandlerManager class is responsible for managing URLSession completion handlers. It stores the completion handlers and provides methods for adding and removing them based on a unique identifier.

Properties

private var completionHandlerQueue: [String: CompleteHandlerBlock]

A dictionary to store URLSession completion handlers with unique identifiers as keys.

Methods

func addCompletionHandler(_ handler: @escaping CompleteHandlerBlock, forIdentifier identifier: String)

Adds a URLSession completion handler to the completionHandlerQueue dictionary with the specified identifier.

func removeCompletionHandler(forIdentifier identifier: String)

Removes the URLSession completion handler associated with the specified identifier from the completionHandlerQueue dictionary.

func callCompletionHandler(forIdentifier identifier: String)

Calls the URLSession completion handler associated with the specified identifier. The handler is removed from the completionHandlerQueue dictionary after it is called.
