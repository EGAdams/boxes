WebFileManager

The WebFileManager class is designed to facilitate the downloading and management of remote files, such as HTML, CSS, and JavaScript files. It is specifically designed to work with WKWebView to display the downloaded files within the app.

Methods

downloadFiles(files:completionHandler:)

This method downloads the specified files from the remote URLs and saves them locally on the device. 

    func downloadFiles(files: [URL: String], completionHandler: @escaping () -> Void)
  • files: A dictionary containing the remote URLs as keys and their corresponding local identifiers as values.
  • completionHandler: A closure that is called once all files have been downloaded and saved locally.

deleteLocalFiles(files:)

This method deletes the specified local files from the device. 

    func deleteLocalFiles(files: [URL: String])
  • files: A dictionary containing the remote URLs as keys and their corresponding local identifiers as values.

prepareWebView() -> URL?

This method prepares the WKWebView to display the downloaded files. 

    func prepareWebView() -> URL?
  • Returns: An optional URL pointing to the local HTML file to be displayed in the WKWebView.

getLocalFileURL(for:) -> URL?

This method returns the local file URL for the given identifier. 

    func getLocalFileURL(for identifier: String) -> URL?
  • identifier: A string representing the local identifier for the desired file.
  • Returns: An optional URL pointing to the local file with the specified identifier.
