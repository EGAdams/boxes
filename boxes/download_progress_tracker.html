DownloadProgressTracker

The DownloadProgressTracker is a class that will handle tracking and reporting the progress of the file downloads. This class will work in conjunction with the Downloader class and provide necessary updates on the download progress.

Responsibilities

  • Track the progress of each individual file download.
  • Calculate the overall download progress based on the individual file download progress.
  • Notify the relevant components (such as UI components) about the download progress updates.

Properties

  • progressUpdateHandler: (Double) -> Void - A closure that will be called with the updated overall download progress (represented as a value between 0 and 1).
  • fileDownloadProgress: [String: Double] - A dictionary that maps file identifiers to their respective download progress values.

Methods

  • updateProgress(forIdentifier identifier: String, progress: Double) - Update the progress of a file download with the given identifier and notify the progressUpdateHandler with the updated overall download progress.
  • resetProgress() - Reset the progress tracking by clearing the fileDownloadProgress dictionary.

Usage

The DownloadProgressTracker will be used by the Downloader class to report the progress of each individual file download. The DownloadProgressTracker will then calculate the overall download progress and notify the relevant components (such as UI components) about the download progress updates.
