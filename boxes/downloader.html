Downloader Class

Description

The Downloader class is responsible for managing the actual download process using URLSession. Its primary responsibilities include creating and managing download tasks, handling background downloads, and resuming interrupted downloads.

Methods

startDownload(url: URL, identifier: String)

Starts a new download task for the provided URL and associates it with the given identifier.

  • url: The URL of the file to be downloaded.
  • identifier: A unique identifier to associate with the download task.

cancelDownload(identifier: String)

Cancels the download task associated with the given identifier.

  • identifier: The unique identifier associated with the download task to be canceled.

resumeDownload(identifier: String)

Resumes a previously interrupted download task associated with the given identifier.

  • identifier: The unique identifier associated with the download task to be resumed.
