PHP File Documentation

This PHP file is responsible for generating a JSON-encoded array containing version and file mapping information. It reads and processes various directories such as templates, JavaScript, CSS, images, and webfonts.

Variables

$config

Reads the "app_config.json" file and stores its content as a string.

$templateDir

Stores the template directory name from the configuration.

$files

An array containing the file paths of all relevant files.

$js, $css, $img, $webfonts

Arrays containing file paths of JavaScript, CSS, images, and webfonts files, respectively.

Functions

ListIn($dir, $prefix = '')

A function that recursively lists all files in a given directory, excluding files and directories with certain conditions, such as the "backup" directory, files with a tilde (~), and sitemap files. It returns an array of file paths.

Main Script

The main script processes the directories, populates the $files array, and then encodes the array into a JSON format, including version and file mapping information, which is then outputted.
