FileHandler Class

The FileHandler class is responsible for managing the file system related tasks, such as saving, deleting, and retrieving files. This class provides an abstraction to handle files on the local device, making it easier to interact with the file system.

Main Responsibilities

  • Save Files: Save files to the local file system with the given URL and directory path.
  • Delete Files: Delete files from the local file system using the provided file URL.
  • Retrieve Files: Retrieve the local file URL based on the given remote URL and directory path.

Methods

  • func saveFile(from remoteURL: URL, to localURL: URL) throws

    This method is responsible for saving a file from a remote URL to a local URL. If an error occurs during the process, it throws an appropriate error.

  • func deleteFile(at url: URL) throws

    This method is responsible for deleting a file at the specified local URL. If an error occurs during the process, it throws an appropriate error.

  • func localFileURL(for remoteURL: URL, in directory: String) -> URL?

    This method is responsible for generating a local file URL based on the given remote URL and directory path. It returns the local file URL if the file exists or nil if the file does not exist.
